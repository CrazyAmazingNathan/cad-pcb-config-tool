LED Controller by Nathan Robb Crazy Amazing Designs – Serial Setup Utility

1. Serial Connection

Disconnected

Use Chrome or Edge on desktop. Plug the controller via USB-C, then click Connect.

2. Wi-Fi / Network Settings

These commands use the board’s serial JSON protocol ({"cmd":"GET_NET"}, {"cmd":"SET_NET"...}). Leave a field blank to keep the current value.

When you press Save Network Settings, the board will attempt to connect with the new values. If it fails within ~30 seconds, it automatically reverts to the previous configuration (as implemented in your firmware).

3. Firmware Update (Web Serial)

This section is for updating the controller’s firmware. For now, this button opens Espressif’s official Web Serial flasher in a new tab. You can select your .bin file there and flash the board directly from Chrome. (Next iteration we can embed esptool-js here for a fully branded flasher.)

Tip: Hold the BOOT/IO0 button on the board (if needed) while connecting for flashing, depending on your ESP32 module’s auto-boot circuitry.
